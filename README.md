# Ecommerce Project
A simple hello world project

# PreRequisites
- nodejs: (version: **v20.11.0**)
- npm: (version: **10.2.4**)
# Quick start
To start this project run the following commands:
```sh
# Install packages
npm install
# Run project
node app.js
```

once server started you can verify by visiting [localhost:3000](localhost:3000)
